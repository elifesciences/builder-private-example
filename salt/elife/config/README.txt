- `etc-certificates-*` files are related to the usage of HTTPS. A wildcard certificate for `*.yourhost.org` or a certificate with Subject Alternative Names should be places here that can be used by all projects (each with its own subdomain). 
- `project-builder-pem` is an SSH private key that must be registered on Github. It will be used for checking out the various public projects through SSH from the virtual machines being created by builder, All the Continuum projects are public, but authentication as *any* Github user is required to use SSH.
